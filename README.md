### Installation / update

1. **Automatic way:**
    - Download install script:  
    `wget -P /tmp https://github.com/sarthakthakur24/poly-light/raw/master/install.sh`
    - Review install script at `/tmp/install.sh`
    - Run it: `bash /tmp/install.sh`
2. **Manual way:**
    - Clone this repository
    ```
    git clone 
    ```
